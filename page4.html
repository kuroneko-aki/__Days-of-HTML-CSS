Introduction to Computing

31.) What is hardware?

  1. Refers to the Physical parts or components of a computer
  2. Stores data and programs permanently
  3. A generic term for organized collections of computer data and instructions

32.) What are Input Devices?

  1. Any peripheral piece of computer hardware equipment to provide data and control signals to an information processing system
  2. Any piece of computer hardware equipment used to communicate the results of data processing carried out by an information processing system
  3. Controls and co-ordinates computer components

33.)What does the abbreviation 'CPU' mean?

  1. Central Protection Unit
  2. Center Protection Unit
  3. Central Processing Unit

34.)What is a CPU??

  1. Used to accomplish specific tasks other than just running the computer system
  2. Responsible for all functions and processes
  3. An extremely fast computer that can perform hundreds of millions of instructions per second

35.)What are Output Devices?

  1. Any peripheral piece of computer hardware equipment to provide data and control signals to an information processing system
  2. Any piece of computer hardware equipment used to communicate the results of data processing carried out by an information processing system
  3. Controls and co-ordinates computer components

36.)What is software?

  1. Refers to the Physical parts or components of a computer
  2. Stores data and programs permanently
  3. A generic term for organized collections of computer data and instructions

37.)What is an example of an Input Device?

  1. Keyboard
  2. Printers
  3. Speakers

38.)What is an example of an Output Device?

  1. Webcams
  2. Monitor
  3. Microphone

39.)What is Random Access Memory (RAM)?

  1. A permanent form of storage
  2. Is responsible for all functions and processes.
  3. A memory scheme within the computer system responsible for storing data on a temporary basis.

40.)What is Read Only Memory (ROM)?

  1. A permanent form of storage
  2. Is responsible for all functions and processes.
  3. A memory scheme within the computer system responsible for storing data on a temporary basis.
